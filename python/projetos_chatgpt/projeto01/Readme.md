```Dashboard Interativo de Vendas```

# Objetivo desse projeto: 
- Desenvolver um dashboard interativo que analise os dados de vendas.


# Métricas utilizadas: 
- Vendas totais
- Vendas por região
- Tendências ao longo do tempo 
- Comparações entre produtos


# Ferramentas: 
- Como o meu foco é desenvolver habilidades usando Python, escolhi utilizar o Python usando a lib do MatplotLib para desenvolver o dataviz deste projeto. 
- (Ferramentas que podem ser usadas) > Tableau, Power BI ou Python (com bibliotecas como Matplotlib, Seaborn e Plotly) para criar visualizações.


# Dataset ( Kaggle ): 
- https://www.kaggle.com/datasets/roopacalistus/superstore


# Informações sobre o dataset 

# Contexto
- O conjunto de dados contém detalhes de vendas de diferentes lojas de uma rede de supermercados que possui várias lojas em diferentes partes dos EUA.
